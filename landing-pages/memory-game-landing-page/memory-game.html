Memory Game

The Memory Game project is a web-based game built using HTML, CSS, and JavaScript. It tests players' memory by requiring them to match pairs of hidden cards. The game includes a stylish user interface, interactive animations, and sound effects to enhance the gaming experience. Additionally, it utilizes local storage to save and display players' high scores, adding a competitive element to the game. This allows players to track their progress and compete against their previous best performances.
