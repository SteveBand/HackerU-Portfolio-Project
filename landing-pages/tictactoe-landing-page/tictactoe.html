Tic-Tac-Toe

Tic Tac Toe is a classic game played on a 3x3 grid. Two players take turns marking X or O in empty cells. JavaScript controls the game logic, HTML defines the structure, and CSS styles the board. The objective is to get three symbols in a row, column, or diagonal.
